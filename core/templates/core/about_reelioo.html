System Architecture v2.0

Hybrid Quantitative
Machine Learning Engine

Reelioo addresses market stochasticity by fusing deterministic technical structure with probabilistic machine learning models.

ABSTRACT: The Reelioo V2 engine operates on a dual-layer architecture. Layer 1 (Deterministic) utilizes a proprietary Quant Engine to define market structure, volatility regimes, and trend persistence. Layer 2 (Probabilistic) employs a Gradient Boosting Machine (XGBoost) trained on 365 days of OHLCV data to detect non-linear patterns invisible to standard indicators. The final "Confidence Score" is a weighted ensemble of these two layers, providing a statistical expectancy rather than a binary signal.

Gradient Boosting Overlay

Traditional algorithms rely on static "if-then" rules. Reelioo utilizes a dynamic Machine Learning Model that ingests a vector of 22 normalized features—including Volume Z-Scores, Wick Rejections, and Relative Strength. It calculates the raw probability of price direction based on historical pattern recognition, allowing the system to adapt to evolving market conditions.

Deterministic Quant Layer

While ML handles prediction, the Quant Engine handles structure. It computes a proprietary Trend Score (-100 to +100) based on exponential moving average slopes and price location relative to key pivots. It strictly categorizes the market into Volatility Regimes using ATR ratios, ensuring that strategies are only deployed in their optimal environment (e.g., Credit Spreads in High Volatility).

The Synthesis Pipeline

01

Feature Extraction

Raw market data (Open, High, Low, Close, Volume) is ingested and transformed into a normalized feature matrix, removing price-scale bias.

02

Dual-Engine Processing

The Quant Engine calculates Trend Strength and Volatility. Simultaneously, the ML Model outputs a raw directional probability (0.0 to 1.0).

03

Probabilistic Weighted Synthesis

The final "Confidence Score" is derived via a weighted ensemble: Score = α(ML_Probability) + β(Trend_Score). This ensures the signal is driven by predictive edge but validated by current momentum.

04

Cognitive Output

The raw numbers are translated into actionable intelligence: Entry Zones, Targets, Dynamic Stop-Losses, and an Institutional "Desk Note" explaining the structure.
